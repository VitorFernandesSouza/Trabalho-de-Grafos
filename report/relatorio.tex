\title{Análise de Interações em Repositórios GitHub usando Teoria dos Grafos}
\author{Vitor (Engenharia de Software - PUC Minas)}
\begin{document}
\maketitle

\section{Introdução}
Este trabalho desenvolve uma ferramenta que representa interações entre colaboradores de um repositório GitHub como grafos direcionados.

\section{Metodologia}
Utilizou-se a API do GitHub para extrair dados de issues e pull requests. Cada interação foi modelada como uma aresta ponderada.

\section{Resultados}
Os grafos gerados foram exportados para o Gephi para análise visual e cálculo de métricas de centralidade e coesão.

\section{Conclusão}
A análise demonstrou como determinados colaboradores atuam como nós centrais e conectores em comunidades do repositório.
